# Codigo de Conduta

O projeto SIGESM Enterprise deve ser conduzido com respeito, clareza e foco
profissional.

## Compromissos

- Tratar colaboradores com educacao.
- Debater ideias tecnicas sem ataques pessoais.
- Registrar decisoes importantes por escrito.
- Manter revisoes objetivas, verificaveis e orientadas a qualidade.

## Comportamentos Inaceitaveis

- Assedio, discriminacao ou intimidacao.
- Comentarios ofensivos ou desrespeitosos.
- Exposicao indevida de dados sensiveis.
- Pressao para contornar controles de qualidade ou seguranca.

## Aplicacao

Problemas devem ser comunicados ao mantenedor do repositorio. Medidas podem
incluir advertencia, remocao de acesso ou rejeicao de contribuicoes.
