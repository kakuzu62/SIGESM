# Roadmap

## Release 0.x Foundation

- Consolidar arquitetura.
- Definir blueprint executivo.
- Criar CI, build e governanca.

## Release 1.x Identity

- Autenticacao.
- Usuarios.
- Criacao de usuarios administrativos.
- Edicao de usuarios administrativos.
- Perfis e permissoes.
- Politicas de senha e sessao.

## Release 2.x Organization

- Cadastro de organizacao.
- Parametros institucionais.
- Integracao com settings.

## Release 3.x Military

- Cadastro completo de militares.
- Restricoes.
- Historico e status operacional.

## Release 4.x Service Scale

- Funcoes de servico.
- Elegibilidade.
- Geracao automatica de escalas.
- Trocas e vendas integradas.

## Release 5.x Desktop Framework

- Shell PySide6.
- Navegacao.
- Formularios e tabelas padronizados.
- Feedback visual e acessibilidade.

## Release 6.x Reports

- Relatorios operacionais.
- Exportacao.
- Consultas de auditoria.

## Release 7.x Build/Deployment

- Instalador.
- Backup e restauracao.
- Atualizacao versionada.
- Pacote de release operacional.
