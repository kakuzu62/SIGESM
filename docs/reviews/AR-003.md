# AR-003 - Architecture Review da STS-001C

## Status

Pendente.

## Checklist

- [ ] Fluxo segue View -> ViewModel -> Application -> Domain -> Repository -> UoW.
- [ ] View nao acessa SQLAlchemy.
- [ ] View nao acessa repository.
- [ ] ViewModel nao acessa SQLAlchemy, session, engine ou Unit of Work.
- [ ] Command contem apenas `user_id`, `full_name`, `username` e `email`.
- [ ] Handler carrega usuario pelo ID.
- [ ] Handler trata usuario inexistente.
- [ ] Handler permite login/e-mail atuais do proprio usuario.
- [ ] Handler rejeita login/e-mail de outro usuario.
- [ ] Handler retorna Result em erros esperados.
- [ ] Agregado preserva ID, senha, estado ativo, roles e `created_at`.
- [ ] `updated_at` e atualizado no sucesso.
- [ ] IntegrityError nao vaza para Presentation.
- [ ] DTO nao expoe senha nem hash.
- [ ] Dialogo de edicao nao exibe campos de senha.
- [ ] AR-001-01 permanece registrado como debito.
- [ ] Ausencia de controle otimista esta documentada.
- [ ] Quality Gate completo executado.
