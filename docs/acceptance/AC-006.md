# AC-006 - Criterios de Aceitacao

- gestao de perfis abre para usuario selecionado;
- perfis disponiveis sao listados;
- perfis atuais sao marcados;
- perfis podem ser atribuidos;
- perfis podem ser removidos;
- duplicidades sao impedidas;
- perfis inexistentes ou inativos sao rejeitados;
- ultimo Administrador ativo e protegido;
- cancelamento nao persiste;
- sucesso atualiza a interface;
- filtro, ordenacao e pagina sao preservados;
- View e ViewModel nao acessam banco;
- DTOs nao expoem dados sensiveis;
- migration esta presente;
- Black, Ruff, MyPy strict e PyTest passam.
