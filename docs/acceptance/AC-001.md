# AC-001 - Criterios de Aceitacao da STS-001A

## Funcionais

- O modulo `Usuarios` abre pelo menu lateral.
- A tabela de usuarios e exibida.
- A pesquisa filtra usuarios.
- A paginacao navega entre paginas.
- A ordenacao por coluna funciona.
- O botao `Atualizar` recarrega a lista.
- O botao `Novo` abre formulario placeholder.
- O botao `Editar` abre formulario placeholder com dados do usuario selecionado.
- O botao `Excluir` permanece desabilitado.

## Arquiteturais

- A View usa apenas `UserListViewModel`.
- O ViewModel usa apenas `UserListingService`.
- A Application usa handlers e contratos.
- A Presentation nao acessa SQLAlchemy.
- A Presentation nao acessa repositories.
- DTOs nao retornam `PasswordHash`.

## Qualidade

- Black deve passar.
- Ruff deve passar.
- MyPy strict deve passar.
- PyTest deve passar.
- Documentacao da STS deve estar atualizada.
