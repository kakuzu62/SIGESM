# AC-003 - Edicao de Usuarios

## Criterios de Aceitacao

- [ ] Usuario selecionado pode ser editado.
- [ ] Nome completo pode ser alterado.
- [ ] Login pode ser alterado.
- [ ] E-mail pode ser alterado.
- [ ] Login duplicado em outro usuario e impedido.
- [ ] E-mail duplicado em outro usuario e impedido.
- [ ] Login atual do proprio usuario e permitido.
- [ ] E-mail atual do proprio usuario e permitido.
- [ ] Senha e hash sao preservados.
- [ ] Estado ativo e roles sao preservados.
- [ ] `created_at` e preservado.
- [ ] `updated_at` e atualizado.
- [ ] Sucesso fecha o dialogo.
- [ ] Falha mantem o dialogo aberto.
- [ ] Listagem e atualizada preservando filtro, ordenacao e pagina.
- [ ] View e ViewModel nao acessam banco diretamente.
- [ ] DTO nao expoe senha nem hash.
- [ ] Black, Ruff, MyPy strict e PyTest passam.
