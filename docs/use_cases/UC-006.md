# UC-006 - Atribuir Perfis a Usuario

## Ator

Administrador autenticado.

## Fluxo Principal

1. O administrador seleciona um usuario na listagem.
2. O administrador aciona "Gerenciar Perfis".
3. O sistema lista perfis ativos disponiveis.
4. O sistema marca os perfis atualmente atribuidos.
5. O administrador altera a selecao.
6. O administrador salva.
7. O sistema valida perfis existentes, ativos e regra do ultimo administrador.
8. O sistema persiste a composicao e atualiza a listagem.

## Fluxos Alternativos

- Cancelar fecha sem persistir.
- Perfil inexistente retorna falha compreensivel.
- Perfil inativo retorna falha compreensivel.
- Remocao do ultimo Administrador ativo e bloqueada.

## Pos-condicoes

- usuario possui exatamente os perfis salvos;
- dados cadastrais e credenciais permanecem preservados;
- a listagem reflete os perfis atualizados.
