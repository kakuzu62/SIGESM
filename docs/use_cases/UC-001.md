# UC-001 - Listar Usuarios

## Ator

Administrador autenticado.

## Objetivo

Visualizar usuarios cadastrados no SIGESM com pesquisa, ordenacao e paginacao.

## Pre-condicoes

- Usuario autenticado.
- Desktop Platform inicializada.
- Modulo `Usuarios` registrado no menu lateral.

## Fluxo Principal

1. O administrador acessa o menu `Usuarios`.
2. O sistema carrega a primeira pagina da listagem.
3. O sistema exibe login, nome, e-mail, status, perfis e ultimo acesso.
4. O administrador pode pesquisar por texto.
5. O administrador pode ordenar clicando no cabecalho da tabela.
6. O administrador pode navegar entre paginas.
7. O administrador pode atualizar a lista manualmente.

## Fluxos Alternativos

### A1 - Nenhum Usuario Encontrado

1. A pesquisa nao retorna resultados.
2. O sistema exibe tabela vazia sem erro tecnico.

### A2 - Parametros Invalidos

1. O ViewModel envia uma query invalida.
2. O handler retorna `Result.failure`.
3. A tela exibe mensagem de erro.

## Pos-condicoes

- Nenhum dado e alterado.
- Nenhuma senha, hash ou token e exposto.
